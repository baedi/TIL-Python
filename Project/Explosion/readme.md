### Segment and Button project

![explosion_img](./explosion.PNG)

#### Description
	- 4자릿수 세그먼트를 이용한 시한폭탄 장난감 구현
	- 지연시간을 활용, 점멸을 이용한 4자릿수 세그먼트 표현
	- 실행시간을 활용, 0초에 가까워질 때마다 피에조 부조의 울림 주기 빨라짐

#### Material
	- Four digit 7-Segment (cathode)
	- Piezo buzzer
	- Wire
	- Resistance 330Ω x 7
